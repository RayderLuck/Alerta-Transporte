# 🚍 Alerta de Transporte Público - Protótipo

## 📌 Objetivo
Criar um aplicativo que avisa em tempo real quando o ônibus ou van está chegando ao ponto, evitando que o passageiro fique esperando sem saber.

## 🔹 Como funciona hoje
- Usa **horários programados** já disponíveis pela Salineira.
- Permite que o usuário escolha seu ponto de ônibus.
- Gera alertas de previsão: "Ônibus previsto em 10 minutos".

## 🔹 Como funcionará com GPS
- Integração com a frota da Salineira.
- Mostra a **localização em tempo real** dos veículos no mapa.
- Envia notificações: "Ônibus a 500m do ponto, chega em 3 minutos".

## 🔹 Benefícios
- Economia de tempo para os passageiros.
- Mais segurança (chegar no ponto só na hora certa).
- Valorização da marca Salineira como empresa moderna e conectada.
- Possibilidade de expansão para vans, transporte escolar e caronas.

## 🔹 Próximos passos
1. Validar protótipo com dados atuais (horários e pontos).
2. Propor parceria para acesso ao GPS da frota.
3. Implementar versão piloto em Cabo Frio.
4. Expandir para toda a Região dos Lagos.
